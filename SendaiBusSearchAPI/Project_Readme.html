ユーザーの ASP.NET アプリケーション

プロジェクトの 作成が完了しました

このアプリケーションの構成:

  • Web API の文書化のためのヘルプ ページ
  • Bootstrap を使用したテーマ
  • 認証を選択した場合、登録とサインインの方法を表示
  • NuGet を使用して管理する ASP.NET 機能

アプリケーションのカスタマイズ

  • ASP.NET Web API を使用して HTTP サービスを使い始める
  • サイトのテーマを変更する
  • NuGet を使用してライブラリを追加する
  • 認証を構成する
  • モデルから ASP.NET Web API のスキャフォールディングを行う
  • Web API をセキュリティで保護する
  • さまざまなデバイスで Web API にアクセスする
  • テストとデバッグのためにトレースを有効にする
  • ASP.NET SignalR でリアルタイム Web を追加する
  • Scaffolding を使用してコンポーネントを追加する
  • プロジェクトを共有する
